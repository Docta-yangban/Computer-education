리스트란 무엇인가?
==================
`<li>` : list를 줄인 것으로, 목차로 표현할 때 필요한 태그입니다.

`<ol>` : ordered list의 약자로, 차례 있는 목차의 부모 태그입니다.
<ol>
    <li>HTML</li>
    <li>CSS</li>
    <li>JavaScript</li>
</ol>

```html
<ol>
    <li>HTML</li>
    <li>CSS</li>
    <li>JavaScript</li>
<ol>
```
<hr>

`<ul>` : unordered list의 약자로, 차례 없는 목차의 부모 태그입니다.
<ul>
    <li>HTML</li>
    <li>CSS</li>
    <li>JavaScript</li>
</ul>

```html
<ul>
    <li>HTML</li>
    <li>CSS</li>
    <li>JavaScript</li>
</ul>
```

이와 같이 표현할 수 있습니다.
